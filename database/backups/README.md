# Бекапы базы данных

Поместите здесь SQL файлы бекапов базы данных PostgreSQL.

Все файлы с расширением `.sql` в этой директории будут автоматически выполнены при инициализации контейнера PostgreSQL.

Файлы выполняются в алфавитном порядке, поэтому рекомендуется использовать префиксы для управления порядком выполнения:

- `01_schema.sql`
- `02_data.sql`
- `03_indexes.sql`

Примечание: Эта директория смонтирована в контейнер PostgreSQL в `/docker-entrypoint-initdb.d/`
